# RandomChoice in Console
2024 reset version of RandomChoice

Use the console to operate, no visual interface.

Compared to the version of winfrom two years ago, this version has more features.

The project was actually written on June 9, 2024, and uploaded on June 25, 2024, a long time ago.

This project was originally written for my own convenience. After thinking about it, I made it public because I thought someone would use it.
